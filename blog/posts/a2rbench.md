# A2RBench: Abstract Reasoning with Formal Verification

Reasoning benchmarks are useful only when the answer can be trusted. For abstract reasoning tasks, manual construction is expensive and uncontrolled generation can introduce ambiguous or unverifiable examples.

**A2RBench** explores an automatic generation paradigm for formally verifiable abstract reasoning benchmarks. The goal is to generate tasks at scale while keeping the correctness signal explicit.

## What makes it valuable

- It separates benchmark generation from informal human intuition.
- Formal verification makes answer checking more reliable.
- Automatic generation can cover broader reasoning patterns.
- The paradigm is useful for evaluating models that claim abstract reasoning ability.
