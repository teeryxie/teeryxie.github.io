Building this homepage

A personal academic page should make the current research direction legible in the first screen, then let visitors drill into papers, projects, notes, and CV details.

This version keeps the stack deliberately simple: static HTML, one CSS file, and GitHub Pages. The structure is organized around research themes, paper artifacts, and project logs rather than generic site filler.

The next useful upgrade is replacing placeholder note entries with actual project logs, release notes, and paper-reading summaries.
